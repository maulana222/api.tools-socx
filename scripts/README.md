# Scripts Backend

Script utilitas satu kali / dev. Jalankan dari folder **backend**:

```bash
node scripts/nama-script.js
```

- **add-isimple-project.js** – Tambah project isimple jika belum ada
- **check-projects.js** – Cek isi tabel projects
- **check-projects-simple.js** – Cek projects (format sederhana)
- **update-isimple-project.js** – Pastikan project isimple ada dan code='isimple'
